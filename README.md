# sruthi-portfolio
# Interactive Portfolio Website

This is my personal portfolio website created to showcase my skills, projects, internships, certifications, and overall learning journey in software development.

The purpose of this project is to reflect my understanding of frontend development, responsive UI design, clean layouts, and basic interactivity using HTML, CSS, and JavaScript.

---

## Features

- Sticky header with smooth navigation
- About section with profile summary and quick facts
- Animated hero section using AOS
- Skills section highlighting technical strengths
- Certifications / achievements gallery with hover effects
- Dark / Light mode toggle
- Responsive design for mobile and desktop
- Resume download section
- Contact section with email and phone details

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- AOS (Animate On Scroll Library)

---

## How to Run the Project

1. Clone or download this repository
2. Open `index.html` in any web browser  
   **OR**
3. View the live website using GitHub Pages
